## 功能特点 ##

* 功能完备，实用导向
* Server端API支持
* 涵盖众多组件、API使用，适用于学习微信小程序
* 多行文本模拟实现
* tab切换
* 模态框
* 本地数据组织及存储
* 图片预览功能

## 使用步骤

1. 克隆代码:

```bash
$ cd 文件夹路径
$ git clone https://github.com/RunningGoteam/RunningGo.git
```

2. 打开`微信Web开放者工具`（注意：须`v0.10.101100`及以上版本）

3. 添加项目

    * AppID：选`无AppID`
    * 项目名称：任意填写
    * 项目目录：
    * 点击 `添加项目`
